Images

  • Hero image width should be 1200px and coded as 600px width.
  • Images should be saved for web in Photoshop.
  • Max image file size should be 250-300 KB.
  • Follow our standard file naming standards:
    Hero Image: img_Hero_[NAME-NO-SPACES]_[01/02/03...]_[WIDTH]x[HEIGHT].jpg
    Porthole Image: img_Porthole_[NAME-NO-SPACES]_[01/02/03...]_[WIDTH]x[HEIGHT].jpg
    Icon: img_Icon_[NAME-NO-SPACES]_[01/02/03...]_[WIDTH]x[HEIGHT].jpg
